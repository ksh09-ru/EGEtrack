# EGEtrack
ExamsTrack - Telegram-бот, созданный на основе ИИ, направленный на помощь школьникам в подготовке к ЕГЭ. Он составляет адаптивный персонализированный учебный план, учитывая расписание и начальный уровень знаний ученика, помогает отслеживать прогресс и экономит время. Доступная альтернатива репетиторам, позволяющая умеренно подготавливаться к ЕГЭ.
